DR. CHIKA EZEUGWU — COMPLETE WEBSITE

WHAT IS INCLUDED
- index.html: complete personal website
- styles.css: responsive design and dark mode
- script.js: mobile menu, animation, theme switcher
- assets/chika-ezeugwu.jpg: portrait extracted from the supplied biography document
- Chika_Ezeugwu_CV.docx: downloadable CV
- codepen.html: HTML-panel copy for CodePen

USING CODEPEN
1. Create a new Pen.
2. Paste the contents of codepen.html into the HTML panel.
3. Paste styles.css into the CSS panel.
4. Paste script.js into the JS panel.
5. Upload the portrait to an image host or CodePen Asset Hosting.
6. Replace assets/chika-ezeugwu.jpg in the HTML with the uploaded image URL.
7. CodePen is ideal for previewing and editing, but your custom domain needs hosting.

LAUNCHING CHIKAEZEUGWU.COM
Option A — Porkbun hosting:
1. Activate Static Hosting or another hosting plan in Porkbun.
2. Upload index.html, styles.css, script.js, the assets folder, and the CV into the public/root directory.
3. Ensure index.html is at the root.
4. Open https://chikaezeugwu.com.

Option B — Netlify:
1. Unzip this package.
2. Drag the entire folder into Netlify's manual deployment screen.
3. Add chikaezeugwu.com as the custom domain.
4. Follow Netlify's DNS instructions in Porkbun.
